# Foobara Rubocop Rules

Just a bunch of linting rules shared across various Foobara projects.

## Installation

Add it to your Gemfile or .gemspec and add the following to your .rubocopy.yml file:

```yaml
inherit_gem:
  foobara-rubocop-rules:
    - rules/*
```
